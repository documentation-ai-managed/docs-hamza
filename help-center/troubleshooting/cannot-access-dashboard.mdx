---
title: Cannot Access Dashboard
description: Fix login and dashboard access issues for Hamza.
---

## Symptoms

- Unable to log in
- Dashboard not loading

## Causes

- Incorrect credentials
- Browser cache problems

## Solution

<Steps>
  <Step title="Check credentials">
    Try resetting your password if unsure.
  </Step>
  <Step title="Clear browser cache">
    Clear history and reload `https://dashboard.hamza.com`.
  </Step>
  <Step title="Try another browser">
    Test access using Chrome or Firefox.
  </Step>
</Steps>
