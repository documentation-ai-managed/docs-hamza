## Overview

To start using Hamza, you'll need to create an account and your first documentation space.

## Steps

1. Visit `https://dashboard.hamza.com`
2. Click "Sign Up" and complete registration
3. Log in to your dashboard
4. Click "New Project"
5. Name your project and save

## Next Steps

Invite team members or start adding docs in your new space.
